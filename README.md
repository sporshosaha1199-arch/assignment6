# 🎯 DigiTools - Premium Digital Tools Marketplace

DigiTools is a modern, high-performance marketplace for premium digital assets, AI tools, and productivity software. Designed for creators and professionals who want to supercharge their workflow.

## 🚀 Technologies Used
- **React.js** (v19)
- **Tailwind CSS** (v4)
- **DaisyUI** (v5)
- **Lucide React** (Icons)
- **Motion** (Animations)
- **React-Toastify** (Notifications)
- **TypeScript** (Type Safety)

## ✨ Key Features
1. **Dynamic Product Catalog**: Browse a curated list of digital tools with detailed features, pricing, and tags (Popular, New, Best Seller).
2. **Interactive Cart System**: Seamlessly add products to your cart, view real-time count updates in the navbar, and manage your selection with a dedicated cart view.
3. **Responsive & Polished UI**: A fully responsive design with smooth animations, sticky navigation, and a modern aesthetic tailored for a premium user experience.

## 🛠️ Functionality
- **Toggling Views**: Switch between the product grid and your shopping cart with a single click.
- **Real-time Notifications**: Get instant feedback when adding or removing items using React-Toastify.
- **Checkout Simulation**: Clear your cart and receive a success confirmation with the "Proceed to Checkout" feature.
- **Smart Cart Logic**: Automatically calculates total prices and provides "Added to Cart" feedback on product buttons.
